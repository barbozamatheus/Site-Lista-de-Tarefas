Site: Lista de Tarefas
Lista de Tarefas criada utilizando HTML, CSS e JavaScript. 

Como um dos meus primeiros projetos, decidi fazer um site que cria uma lista de afazeres a partir de inputs do usuário.
Toda a estrutura foi montada para que o usuário consiga adicionar rapidamente as tarefas que precisa realizar e que consiga excluir alguma delas caso tenha se equivocado ou cumprido.

Pretendo incluir futuramente a função de LocalStorage, para que a lista fique armazenada e não se perca quando a página for atualizada.
